The problem: When you don't specify image dimensions, here's what happens:
  1. Page loads first (text, buttons, etc.)
  2. Image space is empty/invisible
  3. Image finally loads and suddenly appears
  4. Everything else gets pushed around to make room
  5. User experiences jarring page "jump"
solutionAlways include both:
  • HTML attributes: img src="photo.jpg" width="300" height="200"
  • CSS properties: width: 300px; height: 200px;
