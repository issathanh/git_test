There are 3 ways to add CSS styling to your HTML

  1. External CSS
    • Create a seperate file for your CSS
    • Link your HTML using link tag
    • Example:link rel="stylesheet" href="styles.css"
  2. Internal CSS

    put css inside html file (inside of head)

  3. Inline CSS

    put css directly on the html element

    div style="color: white; background-color: black;">Text here/div

    best choice is to use external for whole website
